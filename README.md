## 📖 About

**AcademEcZ** is a lightweight study organizer designed to help students manage their academic life in one place. The goal is to provide a simple, modern interface for organizing subjects, tracking tasks, and improving study productivity.

> **⚠️ This project is still a work in progress.** Some features may be incomplete, experimental, or unavailable.

---

## ✨ Planned Features

* 📚 Subject management
* ✅ Task management
* 📅 Study planning
* 🤖 AI-powered study assistance (Google Gemini)
* 📊 Study progress tracking
* 🔔 Reminders and notifications
* ☁️ Cloud synchronization
* 🌙 Dark mode
* 📱 Responsive design

---

## 🛠️ Tech Stack

* **React**
